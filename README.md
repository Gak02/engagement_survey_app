# ワークエンゲージメント診断アプリ

UWES-9（ユトレヒト・ワーク・エンゲイジメント尺度）を使用したワークエンゲージメント診断Webアプリケーションです。

## 📋 概要

このアプリケーションは、Streamlitを使用して構築されたワークエンゲージメント診断ツールです。9つの質問に回答することで、あなたのワークエンゲージメントを3つのサブスケール（活力、熱意、没頭）と総合スコアで測定し、視覚的に結果を表示します。

## ✨ 主な機能

- **診断機能**: 9つの質問に回答してワークエンゲージメントを測定
- **3つのサブスケール評価**:
  - 活力 (Vigor): 仕事中の高い水準のエネルギーや心理的な回復力
  - 熱意 (Dedication): 仕事への強い関与、意義・熱意・誇りの感覚
  - 没頭 (Absorption): 仕事に集中し、没頭している状態
- **視覚化**: レーダーチャートと棒グラフで結果を可視化
- **結果の解釈**: スコアに基づいた詳細な解釈とアドバイス
- **データエクスポート**: 診断結果をCSV形式でダウンロード可能


### 診断の流れ

1. **診断タブ**: 9つの質問に回答します（各質問は0-6の7段階で評価）
2. **結果タブ**: 「結果を見る」ボタンをクリックすると、診断結果が表示されます
3. **UWESについてタブ**: ワークエンゲージメントとUWES-9についての詳細情報を確認できます

## 📊 スコアの解釈

| スコア範囲 | レベル | 説明 |
|-----------|--------|------|
| 0.0 - 0.9 | 非常に低い | ワークエンゲージメントが非常に低い状態 |
| 1.0 - 2.4 | 低い | ワークエンゲージメントが低めの状態 |
| 2.5 - 3.4 | やや低い | ワークエンゲージメントがやや低い状態 |
| 3.5 - 4.4 | 平均的 | ワークエンゲージメントは平均的なレベル |
| 4.5 - 5.4 | 高い | ワークエンゲージメントが高い状態 |
| 5.5 - 6.0 | 非常に高い | ワークエンゲージメントが非常に高い状態 |

## 📝 UWES-9尺度について

本アプリケーションで使用しているUWES-9（ユトレヒト・ワーク・エンゲイジメント尺度）の質問項目や詳細情報については、論文著者である島津明人研究室の公式サイトからダウンロードできます。

**尺度のダウンロード**: [島津明人研究室 - ワーク・エンゲイジメント (UWES)](https://hp3.jp/tool/uwes)

上記サイトでは、UWES-9の質問項目、使用条件、引用情報などの詳細を確認できます。
